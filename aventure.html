The project consisted of designing an online version of the board game "Les Aventuriers du Rails" using Java as the programming language and a web interface. Its implementation required six months of hard work, during which we had to keep a project report to account for our progress and work details.

However, this project did not go without difficulties. We encountered several obstacles, especially with the complexity of the code and the pressure of the time allotted to successfully complete the project. This required unwavering commitment from all team members. Despite these difficulties, we learned a lot throughout this project. We acquired a solid understanding of the Java language and its use in web development. We also developed our teamwork and organizational skills, learning to communicate better and allocate tasks effectively.

Ultimately, this project allowed us to confront the realities of software development, learning to manage the constraints and challenges that inevitably arise in projects of this magnitude. We are proud of the work we have accomplished and all the skills we have gained during this project.
